# college event management platform
## description : a solution for managing college fests, including registrations,ticketing, and real time updates.

## api's
- register new college fests i.e adding events to it.
- show all / available college fests.
- new user registration / create new account
- login to account
- register for the event and send register success.
- send the success in the form of tickets in the form of pdf.
